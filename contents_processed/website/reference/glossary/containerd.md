 强调简单性、健壮性和可移植性的一种容器运行时


containerd 是一种{{< glossary_tooltip text="容器" term_id="container" >}}运行时，能在 Linux 或者 Windows 后台运行。
containerd 能取回、存储容器镜像，执行容器实例，提供网络访问等。
