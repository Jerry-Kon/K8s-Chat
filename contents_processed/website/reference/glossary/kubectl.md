kubectl 是使用 Kubernetes API 与 Kubernetes 
集群的{{<glossary_tooltip text="控制面" term_id="control-plane" >}}进行通信的命令行工具。


你可以使用 `kubectl` 创建、检视、更新和删除 Kubernetes 对象。
